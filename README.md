# Green Titans ♻️
**Turning Waste into Tokens**

## About the Project
A DePIN (Decentralized Physical Infrastructure Network) of "Digital Waste Exchangers" powered by the Solana blockchain. We transform plastic collection into a transparent process for building social capital.

## Key Features
* **Mobile First:** A dedicated application for citizens and mobile collection units.
* **$SEED Token:** A reward-based economy pegged to the actual weight of recycled materials.
* **Social Ecosystem:** Opportunities to save for education, healthcare, and other social needs.

# Green Titans (RU) ♻️
**Меняем отходы на токен**

## О проекте
DePIN-сеть «Цифровых Обменников Отходов» на базе блокчейна Solana. Мы превращаем сбор пластика в прозрачный процесс накопления социального капитала.

## Основные возможности
- **Mobile First:** Приложение для граждан и мобильных групп сбора.
- **$SEED Token:** Экономика вознаграждений за реальный вес вторсырья.
- **Social Ecosystem:** Накопления на образование, медицину и социальные нужды.
