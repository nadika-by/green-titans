# Архитектура Green Titans

## Технологический стек
- **Blockchain:** Solana (Mainnet-Beta)
- **Smart Contracts:** Rust (Anchor Framework)
- **Mobile App:** React Native
- **Identity:** Интеграция с ИИН (через ZK-proofs)

## Основные компоненты
1. **Oracle Service:** Валидация веса пластика от мобильных групп.
2. **$SEED Mint:** Смарт-контракт для выпуска токенов.
3. **Social Vaults:** Счета целевого накопления (образование, медицина).
